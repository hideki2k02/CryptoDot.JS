What is CryptoDot and how was it made?

CryptoDot was made to simplify your Encryption needs.

CryptoDot was made with HTML, CSS and Node.js with the help of Electron.

The Library i use for Encryption and Decryption is Crypto.JS

The Encryption Methods i use are: AES, SHA-256 and SHA-512

How does the program work?

If you are on Encryption Mode it will try to Encrypt a Node with "Node Name" using the "Node Key".

If you are on Decryption Mode it will do the opposite.

Note for Encryption: If a Node already exists with that name, it will overwrite it!
